## 2.JS网页逆向编程
```
JS的三大块：
    1. ECMA标准语法（最重要）
    2. DOM
    3. BOM
 
 
ECMA标准语法：
1. 解释型语言或者编译型语言
   JS是一个解释型语言，需要解释器，浏览器是解释器
2. 变量
3. 数据类型（基本数据类型，容器类型）
4. 运算符
5. 流程控制语句（分支+循环）
6. 函数
```
